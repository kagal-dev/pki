# @kagal/ct

Certificate Transparency types and schema validators
(RFC 9162). Platform-agnostic — no runtime-specific
dependencies.

## Sub-path Exports

| Export | Description | Deps |
|--------|-------------|------|
| `@kagal/ct/types` | CT interfaces (STH, SCT, proofs, log entries) | none |
| `@kagal/ct/schema` | Valibot validators | valibot |
